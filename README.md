# Messenger Project

This is a full-stack messaging application built with the following technologies:
- **Frontend**: Angular
- **Backend**: ASP.NET Core
- **Cache**: Redis for caching data
- **Database**: MS Sql
- **Containerization**: Docker
## Getting Started

To start the project via Docker, follow these steps:

1. Navigate to the project folder:
   cd messenger/Messenger
2. Build the Docker containers:
docker-compose build
3. Run the Docker containers:
docker-compose up
